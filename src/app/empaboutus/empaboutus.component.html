About HR MANAGER


Webbased App Which Manages Employees


Human Resource Management System project aimed to provide web based HR management tool for all types of organizations. This project includes employee personal details and professional details. Also employee project tracking system. This tool is mainly helping HR of the company in managing all the employee details and salaries. HR can manage leaves records. Company HR which is going to be super user for this tool which is going to have all the permissions for all the employee information including HR details and all records. Employee can see there leave records and can manage their basic information. This HR management can get in touch with the quality control team in Software Company to taking care of employee details provide best quality when developing the company.
